ZeroPads Group Assignment

Welcome to our group assignment. We came up with this project with one goal in mind:
To create a reuseable piece of code to later use in our collaborations with ZeroPads.
After meeting with our sponsor, we decided to move forward with some MLS listings functionality on the ZeroPads website.
So, we created a Real Estate listings application that allows you to search for homes that are for sale by city and state. Head over to our Find Housing tab to start your search.

Technical Know-How

This project was writtin in HTML/ CSS/ JavaScript. It uses the US Real Estate API found on Rapid API. You can find the api details below This project is hosted using AWS S3 Bucket.
