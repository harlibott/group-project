ZeroPads Group Assignment


Welcome to our group assignment. We came up with this project with one goal in mind: to create a reuseable piece of code to later use in our collaborations with ZeroPads. After meeting with our sponsor, we discovered that she wanted MLS listings on her website. So with the intent of that, we created this mock listing application that allows you to search for homes that are for sale by city and state. Head over to our Find Housing tab to start your search.



Technical Details Within The Group Assignment


This project was writtin in HTML/ CSS/ JavaScript. It uses the US Real Estate API found on Rapid API. You can find the api here. This project is hosted using AWS S3 Bucket.
